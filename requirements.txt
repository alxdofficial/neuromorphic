# Core dependencies for neuromorphic LM training.

# The ML/data stack is PINNED to the locally-validated versions (verified 2026-07-11). A floating
# `>=` resolution drifts to unvalidated majors (e.g. transformers 5.13 / datasets 5.0 / numpy 2.5 /
# pyarrow 25) that were never run against — a fresh pod would then train/eval a different stack than
# the one the results were validated on (audit blocker #3). Bump these deliberately, then re-validate.
