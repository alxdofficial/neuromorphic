NOTE (2026-03-28): This file described the old v8 architecture (RL neuromod, port neurons,
7 layers, 1024 neurons). It is now outdated.

Current architecture (v9-backprop, branch v9-backprop):

1. Language Model (Cortical Columns): Split-scan with 4 layers (split at 2), D=2048,
   d_inner=580, GLU output. Lower scan produces H_mid. PCM computes surprise at the
   split point. Split-point MLP combines H_mid + surprise before upper scan. ~52M params.

2. Neural Memory Graph: 512 neurons, D_neuron=256, K=32 connections. Trained end-to-end
   by backprop (differentiable). 2-pass simulation per segment (T=128):
   - Pass 1: ONE gather (frozen initial messages) -> T steps of MLP dynamics
   - Pass 2: ONE gather (from Pass 1 end state) -> T steps refined
   Per-step: state MLP (hidden=24) + message MLP (hidden=24) + neuron_id embedding.
   Segment-boundary modulator MLP (hidden=80) predicts w_conn, decay, primitives.
   Dendritic tree for hierarchical neighbor signal integration.
   Structural plasticity: rewire weakest connections at chunk boundaries.
   ~58M params.

3. I/O: Deterministic inject (replicate H_mid per neuron), parameter-free readout
   (average replicas). No port neurons.

Data flow: tokens -> embedding -> lower scan (2 layers) -> H_mid -> inject into memory
graph -> readout mem_signals -> H_enriched = H_mid + sigmoid(gate) * mem_signals ->
upper scan (2 layers) -> output head -> logits.

Total: ~110M params, ~24.8K tok/s at BS=48 on RTX 4090.
