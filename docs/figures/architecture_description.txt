The Neural Memory Graph architecture has three components operating at different levels:

1. Language Model (Cortical Columns): A scan-based (linear recurrence) LM with 7 layers split into lower (layers 0-3) and upper (layers 4-6) groups, with model dimension D=2048 split across C=16 cortical columns. Input tokens are embedded, projected up to D, then processed by the lower scan layers to produce H_mid. Per-column Predictive Coding Modules (PCM) compute surprise from full context (H, x) with learnable gain modulation.

2. Neural Memory Graph: A persistent network of 1024 neurons with sparse random connectivity (96 presynaptic connections each, key-based softmax routing). At every token, each neuron: (a) receives weighted messages from its 96 presynaptic neighbors (routing weights = softmax(key . neighbor_messages), computed once per segment), (b) integrates them with internal state via exponential decay, and (c) emits a new message through tanh(h * primitives). 16 port neurons interface with the LM - they receive CC signals from H_mid and their outgoing messages become the memory signal. The memory graph is fully persistent across document boundaries. The memory graph runs a sequential per-token loop with a custom Triton kernel.

3. Neuromodulator: An RL-trained policy network (~10M params, 3-layer MLP) that observes all 1024 neurons' states every 128 tokens (514 dims per neuron: primitives, key, mean input, mean output, firing rate, decay) and outputs per-neuron additive deltas (257 dims: delta_primitive, delta_key, delta_decay). Trained via GRPO trajectory scoring: choose K=96 neurons, sample 8 trajectories across all 4 collected chunks, non-K neurons get zero delta, rank by total CE loss, z-score normalize. Best trajectory's final state persists. No value function or critic.

Data flow: input tokens -> embedding -> proj_up -> lower scan (layers 0-3) -> PCM surprise modulation -> H_mid. Memory graph reads H_mid through 16 port neurons, processes per-token dynamics, outputs port neuron messages. Injection: H_enriched = H_mid + sigmoid(gate) * mem_signals. Upper scan (layers 4-6) processes H_enriched. Output: proj_down -> layer_norm -> lm_head -> logits.

The key design principle is that memory is an environment, not a differentiable module. The memory graph is outside autograd - no gradients flow through it. The neuromodulator learns to configure the memory graph via RL, while the LM learns to use memory signals via backprop through the injection gate.
