NOTE (2026-03-29): Architecture description for v9-backprop.

Current architecture (v9-backprop, branch v9-backprop):

1. Language Model: Split-scan with 4 layers (split at 2), D=2048,
   d_inner=580, GLU output. Lower scan produces H_mid.
   PCM predicts transitions (H[t+1]-H[t]), surprise = predicted_delta - actual_delta.
   Split-point MLP combines H_mid + RMSNorm(surprise) before upper scan.
   ~52M params.

2. Neural Memory Graph: 512 neurons, D_neuron=256, K=32 connections. Trained end-to-end
   by backprop (differentiable). 2-pass simulation per segment (T=128):
   - Pass 1: ONE gather (frozen initial messages) -> T steps of MLP dynamics
   - Pass 2: ONE gather (from Pass 1 end state) -> T steps refined
   Per-step: state MLP with structural decay (h = decay*h + (1-decay)*tanh(MLP(input,h)))
   + message MLP (hidden=24) + neuron_id embedding.
   Segment-boundary modulator MLP (hidden=80) predicts w_conn, decay, primitives.
   Dendritic tree for hierarchical neighbor signal integration.
   Phi-based structural plasticity: global percentile prune/grow (2%), 20% exploration.
   Per-step inject/readout (no [BS,T,N,D] materialization).
   ~58M params.

3. I/O: Deterministic inject (replicate H_mid per neuron), parameter-free readout
   (sum / sqrt(N_per_slice)). mem_scale (learnable per-dim) scales readout before
   adding to H_mid: H_enriched = H_mid + mem_scale * mem_readout.

Data flow: tokens -> embedding -> lower scan (2 layers) -> H_mid -> inject into memory
graph -> readout -> H_enriched = H_mid + mem_scale * mem_readout ->
split_mlp(cat(H_enriched, RMSNorm(surprise))) -> upper scan (2 layers) -> logits.

Total: ~110M params, ~24K tok/s at BS=48 on RTX 4090, 18.8 GB peak VRAM.
