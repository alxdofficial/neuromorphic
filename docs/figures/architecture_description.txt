The Neural Memory Graph architecture has three components operating at different levels:

1. Language Model (Cortical Columns): A scan-based (linear recurrence) LM with 7 layers split into lower (layers 0-3) and upper (layers 4-6) groups, with model dimension D=2048 split across C=16 cortical columns. Input tokens are embedded, projected up to D, then processed by the lower scan layers to produce H_mid. Per-column Predictive Coding Modules (PCM) compute inter-column surprise signals with learnable gain modulation.

2. Neural Memory Graph: A persistent network of 1024 neurons with sparse random connectivity (96 presynaptic connections each, L1-normalized weights). At every token, each neuron: (a) receives weighted messages from its 96 presynaptic neighbors, (b) integrates them with internal state via exponential decay, and (c) emits a new message through tanh(h * primitives). 16 port neurons interface with the LM - they receive CC signals from H_mid and their outgoing messages become the memory signal. The memory graph runs a sequential per-token loop with a custom Triton kernel.

3. Neuromodulator: An RL-trained policy network (~10M params, 3-layer MLP) that observes all 1024 neurons' states every 256 tokens and outputs per-neuron modifications to primitives (what neurons broadcast), connection weights (message routing), and decay (temporal persistence). Trained via REINFORCE with a learned value function baseline, collecting rewards across 2 chunks (16 segments) for longer horizon credit assignment.

Data flow: input tokens -> embedding -> proj_up -> lower scan (layers 0-3) -> PCM surprise modulation -> H_mid. Memory graph reads H_mid through 16 port neurons, processes per-token dynamics, outputs port neuron messages. Injection: H_enriched = H_mid + sigmoid(gate) * mem_signals. Upper scan (layers 4-6) processes H_enriched. Output: proj_down -> layer_norm -> lm_head -> logits.

The key design principle is that memory is an environment, not a differentiable module. The memory graph is outside autograd - no gradients flow through it. The neuromodulator learns to configure the memory graph via RL, while the LM learns to use memory signals via backprop through the injection gate.
